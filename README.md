Данный репозиторий используется для фиксации договорённостей, валидации, оценивания по проектам курса ООП C# 


Как пользоваться репозиторием: 
1. Придумайте тему проекта 
2. Создайте новую ветку в репозитории
3. Создайте файл `названиеПроекта.md` в соответствующей директории `год\номер семестра в году`
4. Заполните его в соответствии с `template.md`
5. Создайте PR с упоминанием своего преподавателя практики. Можно дополнительно уведомить преподавателя о созданном PR через другие средства связи 
6. Если у преподавателя будут уточнения/замечания по проекту, то он их оставит комментариями в соответствующем PR. Их нужно будет исправить 
7. Если преподавателя всё устраивает, то он одобряет PR и делает мердж в мастер 

Перед защитой проекта: 
1. Создать новый PR в рамках, которого заполнить недостающие ссылки на презентации и репозиторий проекта 
2. После защиты преподаватель делает аппрув этого PR и вливает его в мастер
