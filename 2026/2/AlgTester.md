### Описание проекта
Движок для симуляции алгоритмической торговли на исторических биржевых данных. 
Позволяет оценивать эффективность различных торговых стратегий без риска реальными средствами.
тип - Web- приложение, для визуализации качественной

### Функциональность
- Считывание исторических графиков цен либо из скачанных CSV-файлов, либо скачивая их на лету через API биржи (например, Binance).
оба варианта, хотим реализовать.


- Настройка пользователем в консоли параметров стратегии, стартового капитала
каждый метод расчёта - реализация интерфейса стратегии, с помощью рефлексии автоматически новые виды стратегий подгружать можно будет. На старте 2-3 алгоритма базовых (пересечение скользящих средних, индекс относительной силы, линии Боллинджера как уже написано выше)
метод расчёта - пошаговая симуляция с учётом настроек от пользователя, сравнение двух стратегий планируем добавить


- Пошаговый пересчёт путём подачи исторических данных выбранному алгоритму, который принимает решения (купить/продать/ждать), а виртуальный кошелек обновляет баланс.


- Вывод статистики по окончании симуляции: сколько было денег, сколько стало, размер чистой прибыли и общее число сделок и тд.
вывод будет визуальный на страничке сайта, графики
аналитический - табличка и например экспорт в виде файла (не обязательно но можно)
